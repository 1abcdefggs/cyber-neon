
# 🚀 CYBER-PINKS Neon Generator

![SVG](https://img.shields.io/badge/Output-SVG-cyan?style=flat-square)

**A professional-grade, browser-based SVG Neon Sign Generator.**  
Create high-fidelity, scalable cyberpunk typography with advanced effects like gas leakage, 3D perspective, and chromatic aberration—all output as pure SVG code.

---

## ✨ Key Features

- 💡 **18 Advanced FX Tools**: Control everything from Halo Glow, Inner Tube Illumination, to realistic Gas Leakage (Noise).
- 🔡 **59 Premium Fonts**: A curated selection of Sci-Fi, Pixel, and Display fonts ready to use.
- ⚡ **Live Presets**: 18 pre-configured styles (Vintage Decay, Glass Reflection, etc.) to kickstart your design.
- 📏 **Relative Scale System**: Typographic scaling based on HTML standards (H1–H6) for professional layouts.
- 🎨 **Dynamic UI Themes**: Switch instantly between **Neon Pink** and **Cyber Cyan** interfaces.
- 📥 **Clean SVG Export**: Download vector files that work perfectly in Figma, Illustrator, or direct web embedding.

---

## 🛠 Usage

1. **Type your text** in the double-bordered input panel.
2. **Select a Font** from the scrollable sidebar.
3. **Choose a Preset** to instantly apply complex filter combinations.
4. **Fine-tune** using the 01-04 FX sliders on the right.
5. **Double-click the Preview** to toggle between standard and expanded zoom view.
6. **Hit DOWNLOAD SVG** to save your creation.

## 🏗 Structure & Contribution

If you want to modify the generator or add new fonts/presets, please refer to our **[Structure Guide](docs/STRUCTURE.md)**.

### File Overview:
- `index.html`: The core UI architecture.
- `main.js`: The Alpine.js logic and SVG filter engine.
- `presets.js`: Data definitions for the 18 advanced style presets.
- `fonts.js`: Library of the 59 supported Google Fonts.
- `style.css`: The Cyberpunk HUD styling system.

### Quick Start for Developers:
```bash
# Clone the repository
git clone https://github.com/1abcdefggs/cyber-neon.git
# Open index.html in your browser (No build step required!)
```
